# Nervespace demo capture

The video shows the public app flow from the home screen to a routine, an active session, completion, progress, and the saved local history entry.

## Capture facts

- Release configuration
- iPhone 17 Pro Simulator on iOS 26.5
- 1206 x 2622 H.264 video at 30 frames per second
- 27.5 seconds with no audio track
- Fixed dark app appearance, captured with the simulator device set to light appearance

To keep the clip under 30 seconds, it uses the app's duration and next controls instead of waiting through every interval. Idle gaps introduced while inspecting the simulator were removed. No screen or completion state was fabricated, and no developer-only or test-only control appears in the clip.
